📦 インストール

NPM でインストール

npm install @naoki85/visual-html-builder

基本的な使い方

import VisualHtmlBuilder from '@naoki85/visual-html-builder'; import '@naoki85/visual-html-builder/dist/index.css'; const editor = new VisualHtmlBuilder('editor-container', { theme: 'default', enabledElements: ['title', 'text', 'image', 'list'] });

CDN から使用

<link rel="stylesheet" href="https://unpkg.com/@naoki85/visual-html-builder@latest/dist/index.css"> <script src="https://unpkg.com/@naoki85/visual-html-builder@latest/dist/index.iife.js"></script> <script> const editor = new window.VisualHtmlBuilder('container'); </script>
